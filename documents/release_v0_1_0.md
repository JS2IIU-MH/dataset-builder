## 概要

v0.1.0 は初期のリリース（プレリリース／最初の安定化）です。本リリースはローカルでのデータ読み込み、基本的なクレンジング、探索的データ解析（EDA）、特徴量エンジニアリング、および Streamlit ベースの簡易UI（チャート表示・エクスポート）を提供します。

## ハイライト

- データ入出力機能の追加（CSV/Excel/Parquet の読み込み）
- データクレンジングの基本ワークフロー実装
- EDA（統計サマリ、相関解析など）の可視化
- 特徴量エンジニアリング用ユーティリティの追加
- Streamlit UI によるチャート表示とエクスポート機能

## 変更点（主なファイル/モジュール）

- `main.py`: アプリ起動エントリ
- `src/logic/data_io.py`: データ読み込み/書き出しロジック
- `src/logic/cleaning.py`: データクレンジング処理
- `src/logic/eda.py`: 探索的データ解析ロジック
- `src/logic/feature_engineering.py`: 特徴量生成ユーティリティ
- `src/ui/charts.py`: 可視化チャート定義
- `src/ui/export.py`: エクスポート機能
- `src/ui/forms.py` / `src/ui/sidebar.py`: UI 入力フォームとサイドバー

（注: 上記は本リポジトリの現状モジュールを概観したものです）

## バグ修正

- v0.1.0 は初期リリースのため、主に機能追加が中心です。既知の小さな不具合については後続のパッチで修正予定です。

## 既知の問題

- 大規模データセットでのメモリ使用量が高くなる可能性があります。
- 一部のファイル形式やエンコーディングで読み込みエラーが発生する場合があります。
- マルチユーザ同時利用を考慮した永続化機能は未実装です。

## アップグレード/移行手順

本リリースからの特別な移行手順はありません。既存データは通常の CSV/Excel/Parquet 形式で問題なく使用できます。

## インストールと実行（簡易）

ローカルで仮想環境を作成し、依存関係をインストールして起動します。

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
streamlit run main.py
```
